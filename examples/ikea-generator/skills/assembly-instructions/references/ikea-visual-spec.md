# IKEA Visual Specification

## Isometric Perspective

- 30-degree isometric angle
- Consistent viewpoint throughout all steps
- Camera distance: medium (show full assembly context)

## Color Palette

| Element | Color | Hex |
|---------|-------|-----|
| Wood parts | Light beige | #E8DCC8 |
| Metal parts | Light gray | #B8B8B8 |
| Screws/fasteners | Dark gray | #4A4A4A |
| Highlight arrows | Red | #E31837 |
| Action indicators | Blue | #0058A3 |

## No Text Rule

Images should contain NO text. All communication through:
- Directional arrows
- Circular highlights
- Dotted lines showing movement
- Explosion views for assembly order

## Hands

When showing hands:
- Generic, non-detailed hands
- Gray/neutral skin tone
- Show grip and tool position clearly
